r/bloks is a subreddit theme built for the new 2018 reddit redesign. It's built to be customizeable and modular, to allow you to set it up in any configuration you like.

You can use the navigation links to pinpoint exactly what you want to change, or you can use the buttons at the bottom of every page to see all of what this theme has to offer.

If you run have any questions be sure to reach out via r/bloks and I'd be happy to help.

If you run into any bugs or issues, you can reach out on r/bloks or you can open an issue on github.
